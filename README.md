# cz4034-IR
Information Retrieval Course Assignment

## Installation
After cloning the repo, run:
```
cd cz4034-IR
npm install
```

## Run Application
Start Solr server:
```
solr-8.4.1/bin/solr start
```
>_To stop the Solr server, run `solr-8.4.1/bin/solr stop`_

Start Node server:
```
npm start
```

Visit the web interface at:
```
http://localhost:3000
```
